# vscale

In order to build and test vscale using the supplied makefile,  
ModelSim must be installed and on the path.  
or verilator must be installed and on the path.

```
cd vscale
make modelsim-sim
make modelsim-run-asm-tests
```
or
```
cd vscale
make modelsim-sim
make modelsim-run-asm-tests
```
